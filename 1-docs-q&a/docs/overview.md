# Acme Cloud Overview

Acme Cloud is a developer platform for hosting APIs and background jobs.

## Products

- **Acme API** — deploy HTTP services
- **Acme Workers** — run scheduled and queue-based jobs
- **Acme Keys** — manage API credentials

## Getting started

1. Create an account at https://acme.example.com
2. Create a project
3. Generate an API key (see API Keys docs)
4. Deploy with the Acme CLI: `acme deploy`
