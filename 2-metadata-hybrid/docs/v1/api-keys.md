# API Keys (v1)

API keys authenticate requests to Acme Cloud.

## Create a key

Email support@acme.example.com with your project ID.
Support creates a key and sends it within 1 business day.

## Reset a key

In v1 there is no self-serve rotate.
Email support@acme.example.com with subject `RESET KEY` and your project ID.
Support revokes the old key and emails a new one.

## Revoke a key

Only support can revoke keys in v1. Open a ticket.
