# API Keys (v2)

API keys authenticate requests to Acme Cloud.

## Create a key

1. Open **Settings → API Keys**
2. Click **Create key**
3. Copy the key once — shown only at creation

Keys look like: `acme_live_xxxxxxxx`

## Reset or rotate a key

1. Open **Settings → API Keys**
2. Select the key
3. Click **Rotate**
4. Update your app with the new key
5. Revoke the old key after the new one works

Rotating does not change your project ID.

## Revoke a key

Revoked keys stop immediately. Create a new key if you still need access.
