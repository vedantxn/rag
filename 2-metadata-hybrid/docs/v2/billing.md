# Billing (v2)

Acme Cloud bills monthly based on usage.

## Plans

- **Starter** — free tier
- **Pro** — pay as you go
- **Enterprise** — custom

## Failed payments

If a payment fails:

1. Update the card under **Settings → Billing**
2. Click **Retry payment**
3. Services stay active for 3 days, then pause

Error code for a declined card is **E_4022**.
